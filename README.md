# Cloud Database Application

A simple web app that connects to a Firebase Firestore cloud database. Users can enter a name and a task that is attatched to that name, then stored in the cloud and can be displayed on the page.

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Download or clone the repository
2. Open the project folder in VScode
3. Right click "index.html" and  and select "Open with Live Server"

Instructions for using the software:

1. Enter a name in the first input field
2. Enter a task in the second user input field
3. Click "Save to Cloud"
4. View the saved entries displayed below (stored in Firebase)

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Visual Studio Code
* Liver Server extention for VScode
* Web Browser (Chrome recommended)
* Firebase Firestore (cloud database)
* Firebase Web SDK (Via CDN)
